# Learn Rust
Rust learning projects following the [The Rust Programming Language](https://doc.rust-lang.org/book/title-page.html) book.

## Set things up
```bash
# installs Rust tooling
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```
